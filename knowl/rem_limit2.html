Remark 4.12.

We can use l'Hopital's rule to find the limit of the sequence \(a_n=\displaystyle\frac{n^2+2}{2n^2+3n}\text{,}\) which is the same as the limit of the function \(f(x) = \displaystyle\frac{x^2+2}{2x^2+3x}\) as \(x\to\infty\text{,}\) since it has the indeterminate form \(\infty/\infty\text{:}\)

\begin{align*} \lim_{x\to\infty} \frac{x^2+2}{2x^2+3x} && \text{Indeterminate }\frac{\infty}{\infty}, \text{ use l'Hopital's rule}\\ &=\lim_{x\to\infty} \frac{2x}{4x+3}& \text{Indeterminate }\frac{\infty}{\infty}, \text{ use l'Hopital's rule}\\ &=\lim_{x\to\infty} \frac{2}{3} = \boxed{\frac{2}{3}}. \end{align*}
