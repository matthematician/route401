Theorem 2.12. Set Algebra Properties.

Let \(A,B,C\) be sets. Then each of the following hold.

\begin{align} A\cup B &= B\cup A && \text{Commutative law for union}\tag{2.7}\\ A\cap B &= B\cap A && \text{Commutative law for intersection}\tag{2.8}\\ A\cup (B\cup C) &= (A\cup B)\cup C && \text{Associative law for union}\tag{2.9}\\ A\cap (B\cap C) &= (A\cap B)\cap C && \text{Associative law for intersection}\tag{2.10}\\ A\cup (B\cap C) &= (A\cup B)\cap(A\cup C) && \text{Distributive law for union}\tag{2.11}\\ A\cap (B\cup C) &= (A\cap B)\cup(A\cap C) && \text{Distributive law for intersection}\tag{2.12} \end{align}
