Let \(A=\{a,b\}\text{.}\) Then we can list all the subsets of \(A\text{:}\)

\begin{equation*} \{a,b\}, \quad \{a\}, \quad \{b\},\quad \emptyset \end{equation*}

so the power set of \(A\) is

\begin{equation*} \mathcal{P}(A) = \bigl\{\, \{a,b\}, \{a\}, \{b\}, \emptyset\, \bigr\}. \end{equation*}
